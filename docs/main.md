# 功能介绍

## 登录

打开APP → 账号 → 教务账号设置，根据提示输入自己在教务系统上设置的账号和密码，并点击获取。

## 首页

系统尝试登录教务系统账号成功后，会获取“个人课表”下的课表信息。

### 日程表

自动适配周目和对应的课程信息。

点击右上角刷新可以手动从教务系统网站更新课表。

点击设置可以自定义课表风格。

调节高度适宜即可一屏展示全天课表。

点击课表上的课程可以查看简明扼要的课程信息（上课教室、教师、班级QQ群号等）。

课表上的 **“调”** 字，提示调课，应当注意调课的时间和地点，自行关注调课的原因。

**当教务系统上有考试信息时，考试信息也将一并自动按照其安排，填充到当天课表上。并冠以“考试”的提示字样。**

## 工具箱

### 班级课表查询

按序修改查询参数，可以查询指定班级相应学期的课表。查询结果也以与首页课程表相同的形式呈现。

### 考试信息查询

按序修改查询参数（默认为当前学期），可以查询自己的考试信息。结果简明扼要，并能缓存在手机中，适宜考试前即用即查。

### 考试成绩查询

按序修改查询参数（默认为当前学期），可以查询自己的考试成绩。

考试成绩按发布时间降序排列，最后发布的在最前面。

**考试成绩不足60分的，在查询结果栏呈红色。** 方便点开瞬间定性看挂科情况。

点开成绩栏，可以查看更丰富的内容（平时成绩得分与比例、成绩发布时间、学分和绩点等）。

### 期末学生评价

支持一键评价。（ToDo）
